# CrowdCount - People Counting Using Video Analytics

This project is a real-time crowd monitoring system using OpenCV.  
It detects and counts people from live video and tracks them in different zones.  

## Features
- Real-time people detection
- Entry and exit counting
- Zone-based monitoring
- Overcrowding alerts
- Data storage for analysis

## Technologies Used
- Python
- OpenCV
- Video Analytics

## Usage
Run the main file to start video processing and counting.
